# Kepler Orbital Model

## Описание проекта
Модель движения планеты по эллиптической орбите с визуализацией в реальном времени.

## Физика
- Второй закон Кеплера: скорость планеты максимальна в перигелии.
- Эксцентриситет орбиты: `e = 0.5`.

## Как запустить
1. Установите Python 3.7+.
2. Установите библиотеки:pip install numpy matplotlib
3. Запустите файл: Планетная система.py
